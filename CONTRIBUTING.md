# Contributing

Thank you for your interest in iHuus!

At this time, we only accept code and documentation contributions from internal iHuus
organization members. External pull requests will be closed without merging.

However, we absolutely welcome your feedback! If you spot a typo, find a bug, or want to
request a feature, please [open an issue](https://github.com/ihuus/api/issues) and our
team will address it.
